Resource Information

Request method GET
Response formats JSON

New Albums

Get the new albums from the market category by area.

Resource URL

http://{{ domain }}/{{ version }}/new/albums/{area}

Resource Information

Request method GET
Response formats JSON
Requires authentication? No

Parameters

area Category by different areas. They are: ALL, ZH, EA, KR and JP.
offset & limit Default: offset = {{ offset }}, limit = {{ limit }}

Examples

  • http://{{ domain }}/{{ version }}/new/albums/JP
