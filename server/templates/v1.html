/search

Search people, song, album, artist, playlist and lyric.

Definition

GET http://{{ host }}/{{ version }}/search

Description

Request GET http://{{ host }}/{{ version }}/search
Response JSON
Authentication No

Params

s The query content.
t
0 people
1 song
10 album
100 artist
1000 playlist
1006 lyric
offset default = {{ offset }}
limit default = {{ limit }}

Examples

  • http://{{ host }}/{{ version }}/search?s=zhao&t=0
  • http://{{ host }}/{{ version }}/search?s=罗大佑&t=1
  • http://{{ host }}/{{ version }}/search?s=罗大佑&t=10
  • http://{{ host }}/{{ version }}/search?s=罗大佑&t=100
  • http://{{ host }}/{{ version }}/search?s=罗大佑&t=1000
  • http://{{ host }}/{{ version }}/search?s=罗大佑&t=1006

Explore playlist by category.

Description

Request GET http://{{ host }}/{{ version }}/explore/playlist/cat/<cat>
Response JSON
Authentication No

Params

cat

情感： 怀旧、清新、浪漫、性感、伤感、治愈、放松、悲伤、温暖、孤独、感动、兴奋、快乐、安静、思念
主题： 游戏、70后、80后、90后、网络歌曲、KTV、经典、翻唱、吉他、钢琴、器乐、儿童、榜单、对唱、00后
