/search

Definition

GET http://{{ host }}/{{ version }}/search

Search people, song, album, artist, playlist and lyric.

Arguments

s The query content.
t
0 people
1 song
10 album
100 artist
1000 playlist
1006 lyric
offset default={{ offset }}
limit default={{ limit }}

Examples

  • http://{{ host }}/{{ version }}/search?s=陈奕迅&t=1
  • http://{{ host }}/{{ version }}/search?s=陈奕迅&t=10
  • http://{{ host }}/{{ version }}/search?s=陈奕迅&t=100
  • http://{{ host }}/{{ version }}/search?s=陈奕迅&t=1000
  • http://{{ host }}/{{ version }}/search?s=陈奕迅&t=1006
  • http://{{ host }}/{{ version }}/search?s=zhao&t=0

/oauth2/tokens

Definition

POST http://{{ host }}/{{ version }}//oauth2/tokens

Request an access token for the user.

Arguments

grant_type A fixed value "password".
username User id or email.
password The password.

Return

ret
-1 not activated
0 email or password is not correct
1 success
access_token The access token for this user.
token_type Default is "bearer".
expire_in Default is 7 days.

Examples

  • curl -X POST http://api.sitfish.com/v1/oauth2/tokens -d "grant_type=password" -d "username=wuzang" -d "password=sss" -v
