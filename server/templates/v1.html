Search

Search users, songs, albums, artists, playlists and lyrics.

Resource URL

http://{{ host }}/{{ version }}/search

Resource Information

Request method GET
Response formats JSON
Requires authentication? No

Parameters

s The query content.
t

1 song
10 album
100 artist
